SETUP - YOU MUST DO THIS BEFORE USING THE PROGRAM OR IT WONT WORK:
1) Double click get-pip.py
2) Double click installlibs.bat

USING THE PROGRAM:
1) Prepare the video you want Jojofied by trimming the video so that the last frame is where you want the arrow to come in
2) Place the video in the 'videos' folder
3) Double click JTBCBot.py
4) Type the number of the file you want to be converted
5) The converted video will be saved in the 'jojofication' folder
